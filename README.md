# 🏁 Carrera de Asteriscos 🏁

Este es un simple juego de consola hecho en Python.

## 🚀 Cómo Jugar

1.  Ejecuta el script `carrera.py` (o como lo hayas llamado).
2.  Apuesta por uno de los 4 corredores (`*`).
3.  ¡Mira la carrera y espera ganar!

## 🔧 Tecnologías Usadas

* Python
* Módulo `random`
* Módulo `os`
* Módulo `time`
